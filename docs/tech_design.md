# Technical Design

## Runtime
- Vanilla JavaScript ES modules
- Single HTML canvas
- Fixed-step simulation at 60 Hz for deterministic automation

## Modules
- `src/config`: difficulty presets and content tables
- `src/core`: constants, utility functions, initial state
- `src/systems`: simulation updates (input, world gen, physics, combat, economy)
- `src/render`: rendering and UI overlays
- `src/main.js`: orchestration and browser integration hooks

## Deterministic Hooks
- `window.render_game_to_text()` returns compact JSON with coordinate system and gameplay state.
- `window.advanceTime(ms)` runs fixed-step simulation and render.

## Content Strategy
- World generated in deterministic chunks around player.
- Chunk seed derives from world seed + chunk coordinates.
- Entities generated by weighted tables in config.

## Testing
- Playwright-based action bursts via skill client.
- Capture screenshots + JSON state snapshots each run.
